# Somebody241 Mylibrewolf

## How do I install it?

Given that the cask is set to be deprecated, i made my own tap for it.

For first time download:
`brew install somebody241/mylibrewolf/mylibrewolf --no-quarantine`

Or `brew tap somebody241/mylibrewolf` and then `brew install --cask mylibrewolf --no-quarantine`.

If already downloaded withouth the '--no-quarantine' parameter, use the necessary command - 'xattr -d com.apple.quarantine /Applications/Librewolf.app' or '/Users/[YourUsername]/Applications/Librewolf.app'
